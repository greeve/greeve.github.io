ATOM

  • https://atom.lib.byu.edu/
  • A Table of Metadata
  • Collections of digitized data
  • Single table of data browsable, searchable on the web
  • Undergraduate students are often employed by librarians to help create and maintain one or more ATOM collections
  • Listing of available collections are listed at the application URL

RS Magazine ATOM Collection

  • http://lib.byu.edu/collections/relief-society-magazine-index/
  • 1914-1970
  • Index of all items in all issues of the magazine
  • Organized by item category (e.g. lessons, articles, fiction, recipes, advertisements, etc.)
  • ATOM records link to scanned version on archive.org
  • Examples
    • rsmag browse: https://atom.lib.byu.edu/rsmag/browse/
    • rsmag search: https://atom.lib.byu.edu/rsmag/search/?q=Alice+Louise+Reynolds
    • rsmag record detail: https://atom.lib.byu.edu/rsmag/17265/
    • rsmag on archive.org: http://www.archive.org/stream/reliefsocietymag39reli#page/4/mode/2up

ATOM API

  • API exposes the data in a way (read-only) that can be programmatically downloaded
  • URL structure: host name, collection slug, action (i.e. search, browse, values), format (i.e. json, csv), query
    • https://atom.lib.byu.edu/<collection_slug>/<action>/<format>/<query>
    • example: https://atom.lib.byu.edu/rsmag/browse/
  • Format output
    • JSON: https://atom.lib.byu.edu/rsmag/browse/json
    • CSV: https://atom.lib.byu.edu/rsmag/browse/csv
  • search output via API
    • https://atom.lib.byu.edu/rsmag/search/json/?q=Alice+Louise+Reynolds
    • https://atom.lib.byu.edu/rsmag/search/csv/?q=Alice+Louise+Reynolds
